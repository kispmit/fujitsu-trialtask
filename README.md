# fujitsu-trialtask

Delivery fee calculator

Java, Spring Boot, H2

Importing weather Data, calculating delivery fee


Spring Boot application, imports weather data and calculates delivery fees based on method of transport and weather situation.

REST API for delivery fee calculation
Weather based pricing
Integration with ilmateenistus.ee
Scheduled weather data importing
H2 database which stores history
Unit tests

Example request:
http://localhost:8080/api/delivery-fee?city=TARTU&vehicle=BIKE
